# Cursor adapter
Copy `wayfinder.mdc` into your project's `.cursor/rules/` directory — it loads on every session (alwaysApply). Adjust the paths inside if Wayfinder lives in a subfolder. Cursor also honors `AGENTS.md`; if your project already uses one, the [`adapters/agents-md/`](../agents-md/agents-md-block.md) block works there too — use one or the other, not both.
